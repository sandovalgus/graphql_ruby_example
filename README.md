# README
Ejemplo de una API utilizando Graphql y siguiendo el tutorial de howtographql ( https://www.howtographql.com/graphql-ruby/0-introduction/ ).

## Installation

Install dependencies:

```
bundle install

rails db:create
rails db:migrate
rails db:seed
```

Starting the server:

```
rails server
```

Opening the application:

```
open http://localhost:3000/
```
